[ 반복문 ]
    1. 조건을 충족할때 특정 범위 내 코드를 반복 실행 
    2. for 형태
        for(초기값; 조건문;증강식){
            실행문;
        }
        1.초기값 : 반복을 할때 반복의 카운터 하는 변수의 처음값
        2.조건문 : 반복실행문이 실행되는 조건으로 true 이면{실행문} 처리하고 false리면 
        3.실행문 : 조건이 true 일때 실행할 코드
        4.증강식 : 반복변수의 증가 혹은 감소 변화 연산식

    3.반복문 연습하는 방법
        *추천*:반복문 없이 코드를 작성하고 반복되는 코드와 반복되지 않는 코드, 패턴/규칙 찾기

    4.배열과 반복문 관계
        1. 배열은 인덱스가 0부터 마지막 인덱스 1씩 증가 규칙
        2. 배열의 길이 : 배열 변수명.length,
            -> 배열내 마지막인덱스 : 배열변수명.length -1
        3. 배열의 순회/전체조회 : 배열내 모든 요소를 하나씩 호출하는 작업
        for(let index=0;index<=배열변수명.length-1;index++){
                배열변수명[index]
        }

    5. 반복문의 주요 키워드
        1. continue : 가장 가까운 for문의 증감식으로 코드흐름 이동한다.
        2. break : 가장 가까운 for문의 {} 종료/탈출/끝내기
