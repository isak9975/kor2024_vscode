    2. 함수 종류
        1. 내가 만든 함수 : 개발자가 함수를 직접 코드로 정의/사용 한 함수
        2. 미리 만들어진 함수(라이브러리) : 개발자가 미리 만들어진 함수를 사용
            -console.log(), document.querySelector(),push(),splice() 등등

    3. JS 함수 표현 방법
        1. 선언적 함수 : 선언 키워드를 이용하여 함수르 정의하는 방법, function함수명(){}
        2. 익명 함수 : 함수명이 없이 정의하는 방법, function(){}, let 변수명 = function(){}
        3. 람다식(화살표) 함수 : 선언키워드와 함수명 없이 -> 화살표 정의하는 방법, () => () let 변수명 = () => ()

    4. JS 함수 만드는 방법
        1. function         : 함수를 만들기 위한 함수 선언 키워드
        2. 함수명            : 관례적으로 카멜 표기법 사용 mycar -> myCar
                            -카멜표기법이란? 첫 단어를 소문자로 시작하고 두번째 단어의 첫글자를 대문자 표현
        3. (매개변수1,매개변수2) : 함수 사용시 함수 안으로 들어오는 인수값을 대입받는 변수
            -인(들어갈 인)수(숫자 수)
        4. {명령어;}             : 함수가 실행 될때 처리할 명령어들의 구역
        5. return                : 함수가 처리하고 함수를 호출했던 곳으로 반환 되는 데이터, 리턴값 ,반환값

    5. JS함수 사용/호출 하는 방법
        정의한함수명(인수값,인수값)     :  인수값이 각 함수의 매개변수에 대입되고 함수 실행후 함수의 return 값이 반환 된다.

    6. JS함수 특징

        2. 미리 정의된 코드의 매개변수에 따른 서로 다른 결과(리턴) 없을 수 있다.
        3. 협엽에서 기능단위로 구분 할수 있다.
